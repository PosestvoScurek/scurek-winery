Nastavitev povezave

Za prenos naročil v realnem času potrebujete brezplačen Supabase račun. Enkratna nastavitev, nato vse deluje avtomatsko.

1
Pojdite na supabase.com → New Project → ustvarite projekt
2
V projektu: Settings → API → kopirajte Project URL in anon key
3
V SQL Editor zaženite: CREATE TABLE orders (id uuid default gen_random_uuid() primary key, created_at timestamptz default now(), items jsonb, location text, room text, note text, total numeric, status text default 'new');
Napaka — preverite URL in ključ.

Podatki se shranijo samo v vašem brskalniku.
